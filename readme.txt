*To Compile this program clone the repository and run the following command
	g++ -std=c++11 -W -Wall main.cpp
	
/**************************************
Compiled executable is also availiable in exe folder.

This program basically creates the 2 types of shapes (triangle and circle) which is composed of points 
and draw them on the Graphical Context which is the terminal in our case.

Class Triangle and Circle is composed of Class Points and inherits from Class Shapes.
